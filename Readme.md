# 🛒 ShopWave Agentic AI

An autonomous customer support system built using Django + Anthropic API.

## 🚀 Features
- Tool-based reasoning (Agentic AI)
- Fraud detection
- Policy-based decisions
- Ticket automation

## 🧠 Tools
- get_customer
- get_order
- check_refund_eligibility
- cancel_order
- escalate_ticket
- search_knowledge_base

## ⚙️ Setup

pip install -r requirements.txt
python manage.py runserver
